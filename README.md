# Pasos para correr el proyecto

**Crear variables de entorno**
    Crear archivo .env tomando .env.example como ejemplo

**Crear entorno**
    `py -m venv env`

**Activar entorno**
    `.\env\Scripts\activate`

**Migrar base de datos**
    `py manage.py migrate`

**Correr app**
    `py manage,py runserver`
